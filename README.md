# SQL for Data Analysis: Advanced SQL Querying Techniques-Udemy
This repository contains assignments from the SQL Udemy course. It covers a wide range of SQL concepts and real-world data analysis techniques.

List of Topics Covered
- Joins & Unions
- Subqueries & CTEs
- Window Functions
- Numeric, String & Date Functions
- Null Handling & Filtering
- Duplicate Management
- Aggregations & Min/Max Functions
- Pivoting
- Rolling Calculations
